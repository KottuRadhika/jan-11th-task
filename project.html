MY PROJECT

A sentimental Analysis System To Improve Teaching And Learning
Using Django



ABSTRACT: The main of the projectis to analyse the sentiments from the student feedback system and show the user's attitude towards a particular subject.NLP and machinelearning can be applied to student feedback to help university administrators and teachers address problematic areas in teaching and learning. The proposed system analyzes student comments from both course surveys and online sources to identify sentiment polarity, the emotions expressed, and satisfaction versus dissatisfaction. A comparison with direct-assessment results demonstrates the system's reliability.
